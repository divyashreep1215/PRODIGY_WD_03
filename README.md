# 🎮 Tic-Tac-Toe Web Application

A simple and interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript.  
This project demonstrates core front-end development concepts and game logic implementation.

---

## ✨ Features

- Player vs Player mode
- Player vs AI mode
- Animated winning line
- Draw detection
- Restart game without refreshing
- Responsive design
- Clean and modern UI

---

## 🧠 Game Logic

- Tracks board state using an array
- Alternates turns between X and O
- Prevents overwriting filled cells
- Checks winning conditions (rows, columns, diagonals)
- Detects draw situations
- Disables board after game ends

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Grid & Animations)
- JavaScript (DOM Manipulation)
